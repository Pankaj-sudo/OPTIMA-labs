How to grant admin access
curl -X POST https://asia-east1-optima-labs.cloudfunctions.net/setAdminClaim \ -H "Content-Type: application/json" \ -H "x-admin-secret: <ADMIN_SECRET>" \ -d '{"email":"new-admin@gmail.com"}'

Admin access is granted by the Firebase custom claim {admin:true}, not by this list. The user must sign in once, then sign out/in after the claim is set.
